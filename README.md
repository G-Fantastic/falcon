# 📚falcon
Common libraries for Java

## 🛠️Environment（开发环境）
+ JDK 9.0.4
+ Apache maven 3.6.1


## 💿集成方式（兼容Java 8及Java 9+）
### Maven
```xml
<dependency>
  <groupId>com.iofairy</groupId>
  <artifactId>falcon</artifactId>
  <version>0.3.2</version>
</dependency>
```

### Gradle
```
implementation 'com.iofairy:falcon:0.3.2'
```
